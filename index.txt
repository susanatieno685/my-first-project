GIT STAGES:
1. Working stages; actual project,files you are currently editing
ADD /create file
deleting a file
edit the comment of a file
2. Staging area;preparation zone
Git takes the snapshot of the file
git add
git add .    -> adds all the files in the working area
git add <name of the file>    -> adds the specific file
git add *.html   -> adds all files with the extension
3. Repository;Push final work
git commit -m "commit message"
(git takes a snapshot of the work and are stored with a changes id and git message
